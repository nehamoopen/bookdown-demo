Guide to Data Privacy

Chapter 1 Background to the Project

The aim of this project is to build up knowledge, tools, and experience of how researchers can and should deal with personal and sensitive data, with an emphasis on pseudonymization and anonymization techniques + tools. At the moment, knowledge about this appears limited or at best, scattered. This is because different disciplines or projects encounter different issues and/or approach from different angles. This UU-wide project aims to collect information and experiences and build a knowledge base and tool base for UU researchers, wherein the solutions are accessible and reusable as far as possible.

The specific project goals are as follows: 1. Create a knowledge base for researchers to learn about pseudonymization and anonymization. 2. Create a tool repository for researchers. 3. Realize or expand a number of tools to solve concrete problems of research projects in this area. 4. Develop various workshops and training courses for researchers and support staff about dealing with privacy-sensitive data and the use of the available tools.
